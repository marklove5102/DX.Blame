  <action>
Rewrite DX.Blame.Hg.Provider.pas to replace all six ENotSupportedException stubs with real Mercurial CLI implementations. Follow the TGitProvider delegation pattern exactly.

**Changes to the unit:**

1. **Update unit header:** Change "Stub IVCSProvider" to "Full IVCSProvider implementation for Mercurial" in the summary/remarks.

2. **Move constants to Hg.Types:** Remove the `cHgUncommittedHash` and `cHgNotCommittedAuthor` const declarations from the implementation section (they now live in DX.Blame.Hg.Types).

3. **Update uses clause (implementation):** Add `DX.Blame.Hg.Types, DX.Blame.Hg.Process, DX.Blame.Hg.Blame` alongside existing `System.SysUtils, DX.Blame.Hg.Discovery`.

4. **Update GetUncommittedHash:** Now returns `cHgUncommittedHash` from Hg.Types (40 f's instead of the old 12 f's).

5. **Implement ExecuteBlame:**
   - Create THgProcess with `FindHgExecutable` and `ARepoRoot`
   - Compute relative path: `ExtractRelativePath(IncludeTrailingPathDelimiter(ARepoRoot), AFilePath)` then `StringReplace(LRelPath, '\', '/', [rfReplaceAll])`
   - Call `LProcess.ExecuteAsync(BuildAnnotateArgs(LRelPath), AOutput, AProcessHandle)`
   - Return the exit code
   - Free the process in a try/finally

6. **Implement ParseBlameOutput:**
   - Delegate to `DX.Blame.Hg.Blame.ParseHgAnnotateOutput(AOutput, Result)`
   - Single line, same pattern as TGitProvider.ParseBlameOutput

7. **Implement GetCommitMessage:**
   - Create THgProcess with FindHgExecutable and ARepoRoot
   - Execute `'log -r ' + ACommitHash + ' -T "{desc}"'`
   - On exit code 0: `AMessage := Trim(LOutput)`, return True
   - Otherwise return False
   - Free process in try/finally

8. **Implement GetFileDiff:**
   - Create THgProcess with FindHgExecutable and ARepoRoot
   - Execute `'diff -c ' + ACommitHash + ' "' + ARelativePath + '"'`
   - On exit code 0: `ADiff := LOutput`, return True
   - Otherwise return False

9. **Implement GetFullDiff:**
   - Create THgProcess with FindHgExecutable and ARepoRoot
   - Execute `'diff -c ' + ACommitHash`
   - On exit code 0: `ADiff := LOutput`, return True
   - Otherwise return False

10. **Implement GetFileAtRevision:**
    - Create THgProcess with FindHgExecutable and ARepoRoot
    - Execute `'cat -r ' + ACommitHash + ' "' + ARelativePath + '"'`
    - On exit code 0: `AContent := LOutput`, return True
    - Otherwise return False

**Discovery methods remain unchanged** (FindExecutable, FindRepoRoot, ClearDiscoveryCache, GetDisplayName, GetUncommittedAuthor already work).

File MUST be UTF-8 with BOM and CRLF line endings.
